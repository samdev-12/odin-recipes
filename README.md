# Odin Recipes

Este proyecto es parte del curso de fundamentos de HTML de The Odin Project. El objetivo es practicar la estructura básica de páginas web usando solo HTML, sin estilos ni frameworks.

## Sobre el proyecto

El sitio contiene una página principal con enlaces a varias recetas simples. Cada receta está separada en su propio archivo HTML e incluye:
- Una imagen del plato
- Una descripción corta
- Lista de ingredientes
- Instrucciones paso a paso

Las recetas incluidas son:
- Pizza
- Hot dog
- Sándwich

## Sobre mí

Este proyecto fue hecho mientras aprendía desarrollo web desde cero. Estoy empezando a entender cómo funciona Git, GitHub, la estructura de proyectos y cómo organizar archivos correctamente en una carpeta de desarrollo.

También estoy usando herramientas como VS Code (o VSCodium) y Git desde terminal en Linux/WSL para manejar mis proyectos.

## Notas

A veces el proyecto se siente más complicado de lo que parece, especialmente con rutas de archivos, imágenes y Git, pero es parte del proceso de aprendizaje.

Este repositorio es una práctica para mejorar mis habilidades básicas antes de pasar a CSS y proyectos más complejos.
